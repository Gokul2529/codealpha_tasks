# Fitness Tracker App

A simple fitness tracking web app that lets users log daily fitness activities and view progress using local storage.

## Features

- Add daily entries for steps, workout minutes, and calories burned.
- Dashboard summary for today's totals.
- Weekly progress bars for the last 7 days.
- Saves entries in browser localStorage.

## How to use

1. Open `index.html` in your browser.
2. Add a new entry using the form.
3. Review the daily summary and weekly progress.
4. Entries persist automatically in local storage.

## Files

- `index.html` — main app structure
- `styles.css` — app styling
- `app.js` — data handling and local storage logic
