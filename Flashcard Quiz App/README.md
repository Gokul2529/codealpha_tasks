# Flashcard Quiz App

A lightweight flashcard quiz app built with HTML, CSS, and JavaScript. It lets you review cards, reveal answers, and mark whether you remembered each one.

## Features

- View a sequence of flashcards
- Tap the card to reveal the answer
- Mark cards as correct or incorrect
- Add new custom flashcards
- Keep track of progress and score

## Usage

1. Open `index.html` in your browser.
2. Click or tap the flashcard to reveal the answer.
3. Use the buttons to navigate cards and mark your responses.
4. Add new flashcards with the form at the bottom.

## Files

- `index.html` — App layout and structure
- `styles.css` — Styling for the quiz app
- `app.js` — Flashcard quiz logic
